Getting Started with Akhfhid Bot

Follow these simple steps to get Akhfhid Bot up and running in your server.

  1. Invite the Bot: Click the "Invite to Server" button on the home page and select your server.
  2. Grant Permissions: Make sure the bot has the necessary permissions to read and send messages, connect to voice channels, etc.
  3. Run a Command: Try typing !help in any channel to see the list of available commands.
  4. Explore Features: Check out the music, moderation, and fun features to make your server more engaging!

Command List

Here are all the commands you can use with the Akhfhid bot:

AI & Utility Commands

  • !chat [message] — Chat with Akhfhid AI.
  • !airesearch [topic] — Perform in-depth research using AI.
  • !plex [query] — Search for information with Perplexity AI.
  • !roblox [username] — Check complete Roblox user information.
  • !help — Displays a list of all available commands.
  • !ping — Check the bot's response speed.
  • !reqfitur [message] — Send a feature request to a dedicated channel.

Leveling Commands

  • !leaderboard — Displays the top 10 users with the highest level in the server.
  • !level [@user] — View your or another user's level and XP.

Music Commands

  • !play [song name] — Play a song or add it to the queue.
  • !queue — Displays the music queue.
  • !skip — Skips the currently playing song.
  • !stop — Stops the song and clears the queue.

Admin & Moderation Commands

  • !maintenance [message] — Send a maintenance announcement to a specific channel.
  • !maintdone [message] — Send an announcement that maintenance is complete to all servers.
  • !setschedule — Configure the daily schedule channel and status.
  • !welcome [#channel] — Configure the server's welcome message channel.

Information Commands

  • !serverinfo — Shows information about the server.
  • !userinfo [@user] — Displays detailed information about a user.

Frequently Asked Questions (FAQs)

Q: The bot is not responding to my commands.

A: First, check if the bot has the correct permissions. If it does, try kicking and re-inviting the bot to your server. This often resolves permission issues.

Q: How can I change the bot's prefix?

A: Currently, the prefix is fixed. However, we are working on a customizable prefix feature that will be available in a future update.

Q: Can I suggest a new feature?

A: We appreciate your feedback. Use the command !reqfitur to submit your feature request directly to the owner.
